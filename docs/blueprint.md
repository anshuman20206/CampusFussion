# **App Name**: CampusConnect

## Core Features:

- Homepage Hero Section: Display a hero section with a compelling heading, subtext explaining CampusConnect’s mission and GDG affiliation, and prominent 'Join' and 'Explore Events' buttons to guide user engagement.
- AI Assistant: Integrate Gemini AI to provide doubt-solving in programming (Firebase, Java, React, Python), offer event information, provide resources for GDG events, assist with resume and interview tips, give tech stack advice, and connect students to mentors and leads. The AI tool will decide when the provided content and/or links might be helpful to the user.
- Chatbot Interface: Create a clean chat interface with user/bot chat bubbles, transitions, and a 'Thinking...' typing indicator.
- Events Display: Display events fetched from Firebase Firestore in a timeline view. Include event name, date, description, location, and a registration link.
- Photo Gallery: Showcase event photos in a grid, allowing users to preview images in a modal.
- Announcements: Present admin-posted announcements, highlighting pinned announcements and displaying them by date, with the most recent announcements at the top.
- Community Links: Provide links to CampusConnect's Discord, Telegram, and LinkedIn communities using cards and member highlights to encourage user participation.

## Style Guidelines:

- Primary color: Google Blue (#4285F4) to align with the Google brand and create a sense of trust and innovation.
- Background color: Light gray (#F9FAFC), near-white, providing a neutral and clean backdrop that enhances readability and focuses attention on content.
- Accent color: Google Yellow (#FBBC05), adding vibrancy and drawing attention to important interactive elements.
- Body and headline font: 'Inter', a sans-serif font known for its legibility and modern appearance. Note: currently only Google Fonts are supported.
- Minimalist and responsive layout to ensure the platform is accessible and visually appealing across all devices.
- Subtle animations and transitions using AOS.js, enhancing user experience with smooth content loading and navigation.
- Follow Google's Material Design principles in icon design for a consistent, user-friendly, and recognizable look and feel.
